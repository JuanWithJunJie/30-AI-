# 30天AI术语科普营

面向制造业场景的AI入门学习资料仓库，包含30天系统化的AI术语科普内容。

## 内容概览

### 核心课程（Day01-Day30）

以HTML卡片形式呈现的30天AI学习内容，覆盖：

| 主题 | 涵盖内容 |
|------|----------|
| 基础概念 | Token、Prompt、上下文窗口、LLM |
| 模型技术 | 多模态、训练微调、推理、Embedding、RAG |
| 应用技术 | Agent、Skill、MCP、向量化 |
| 行业应用 | AIGC、CV、NLP、智能制造、预测性维护、AI质检 |
| 部署方式 | 端侧AI、云侧AI |
| 前沿话题 | AI幻觉、AGI、参数规模 |

### 案例分类

课程包含30+个工厂实际案例，涵盖：
- **质量管理**：AI质检、良率分析、缺陷检测
- **设备维护**：预测性维护、知识库检索
- **生产流程**：ERP填报、多智能体协作
- **工厂制造**：Agent应用、LLM选型

## 使用方式

1. **在线阅读**：直接在浏览器中打开 `DayXX_*.html` 文件
2. **案例查询**：通过 `AI案例分类汇总.md` 快速定位相关案例
